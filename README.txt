# GBO Trainer Tool – PWA-Version

Diese Version kann als Web-App genutzt und auf dem Handy zum Home-Bildschirm hinzugefügt werden.

## So nutzt du sie
1. Ordner auf einen Webserver hochladen, z. B. eigener Webspace, Netlify, Vercel oder GitHub Pages.
2. `index.html` öffnen.
3. Am Handy im Browser öffnen.
4. iPhone: Teilen → Zum Home-Bildschirm hinzufügen.
5. Android: Menü → App installieren / Zum Startbildschirm hinzufügen.

## Enthalten
- index.html
- manifest.webmanifest
- service-worker.js
- App-Icons

## Wichtig
Die App funktioniert als PWA richtig, wenn sie über HTTPS oder localhost geöffnet wird. Direkt als lokale Datei ist die Installation meistens nicht möglich.
